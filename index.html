1. What is an Array?

An array is a data structure used to store multiple values of the same data type in a single variable. Instead of creating many separate variables (a, b, c), you use one.


Real-Life Analogy ü•ö

Think of an array like an egg tray:

  • üîπ Tray slots: Represent the index.
  • üîπ Eggs: Represent the elements.
  • üîπ Same type: You only store eggs (not rocks).
  • üîπ Fixed size: A tray has a fixed number of slots.

Why Use Arrays?

  • ‚úÖ Store Many Values: Keep related data together.
  • ‚úÖ Fast Access: Random access using index arr[i].
  • ‚úÖ Clean Code: Avoid chaos of variables.
  • ‚úÖ Sequential Memory: Optimized for CPU Cache.

Sorting Algorithms

Organizing data is crucial. Let's visualize Bubble Sort, a simple algorithm that repeatedly swaps adjacent elements.

Bubble Sort Code O(N¬≤)

for(int i=0; i<n-1; i++) {
    for(int j=0; j<n-i-1; j++) {
        if(arr[j] > arr[j+1]) {
            swap(arr[j], arr[j+1]);
        }
    }
}

Two Pointer Technique

A powerful strategy to solve problems efficiently (often O(N)) by using two indices usually moving towards each other.

Pattern: "Meet in Middle"

Used for reversing arrays or checking palindromes.

int left = 0, right = n-1;
while(left < right) {
    process(arr[left], arr[right]);
    left++; 
    right--;
}

Example: Pair Sum (Sorted)

Find two numbers that add up to target.

// Array MUST be sorted
while(L < R) {
    int sum = arr[L] + arr[R];
    if(sum == target) return true;
    else if(sum < target) L++;
    else R--;
}
